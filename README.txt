To run the code:
-> Make sure the correct libraries are installed (pyo most notably).
-> Run musiclib.py.
-> The settings have all been set to default so you merely need to press enter to get through the settings up until music starts being played.
-> When the graph comes up, you merely need to exit it and press Y to continue.
